# **_Output Dalam ALGORITMA_**
Output dalam Algoritma
Dalam dunia algoritma, output adalah hasil atau informasi yang dihasilkan oleh algoritma setelah memproses input. Output ini merupakan solusi atau jawaban dari masalah yang ingin dipecahkan oleh algoritma. Output bisa berupa angka, teks, gambar, atau jenis data lainnya, tergantung pada tujuan dari algoritma tersebut.

# Contoh
Masih menggunakan contoh algoritma sederhana untuk menghitung luas persegi panjang, setelah menerima input berupa panjang dan lebar, algoritma akan menghasilkan output berupa luas persegi panjang.  Luas ini dihitung dengan mengalikan panjang dan lebar.  Jadi, output adalah hasil perhitungan tersebut.

# Output dalam Python
Dalam bahasa pemrograman Python, output biasanya ditampilkan kepada pengguna melalui fungsi **print()**. Fungsi ini memungkinkan program untuk menampilkan data atau informasi ke layar. Output ini bisa berupa teks, nilai variabel, hasil perhitungan, atau pesan lainnya.

# Contoh
nama = input("Masukkan nama Anda: ")
print("Halo,", nama, "!")
